## PYTHON CONTROL STRUCTURES AND DATA TYPES PRACTICE

#### Overview

This repository contains 15 standalone Pythone scripts to practice:

- Conditional statements
- Loops
- Data types and type checking
- Collections
- User input handling and type casting

Each script is runnable independently and focuses on a specific concept.

### Scripts

1. Age Access Program – Grants access based on age.
2. Admission Fee Calculator – Calculates fees by age.
3. Even or Odd Analyzer – Checks numbers 1–10 for parity.
4. Grade Classifier – Classifies test scores.
5. Discount Program – Applies discounts based on price.
6. Login Simulation – Validates email/password.
7. Weekend Detector – Maps numbers 1–7 to weekdays.
8. Collection Sorter – Prints numbers >10 and counts them.
9. Favorite Fruits – Prints all color–fruit combinations.
10. Membership Checker – Checks if a car type exists.
11. Dictionary Access Validation – Updates verification status.
12. Login Attempts – Limits password attempts to 3.
13. Type Identifier – Detects data type of input.
14. Score List Evaluator – Counts scores above/below 70.
15. Nested Data Lookup – Prints family member info from a nested dictionary
